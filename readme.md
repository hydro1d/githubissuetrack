1. What is the difference between var, let and const?
Answer: var is the legacy method for declaring variables; it can be reassigned and redeclared. let is the modern approach, allowing reassignment but preventing redeclaration in the same scope. const is used for values that must remain constant after initial assignment.

2. What is the spread operator?
Answer: The spread operator (...) expands arrays and objects into individual elements, enabling easy duplication and merging of collections.

3. What is the difference between map(), filter() and forEach() loop?
Answer: map() transforms each element and returns a new array with the transformed values. filter() returns a new array containing only elements that satisfy a specified condition. forEach() iterates through elements for side effects without creating a new array.

4. What is an arrow function?
Answer: An arrow function is a concise syntax for writing functions in JavaScript using the => notation, often requiring less code than traditional function declarations.

5. What are template literals?
Answer: Template literals are strings enclosed in backticks that allow variable interpolation using ${} syntax, enabling cleaner and more readable string formatting.
