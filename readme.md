# 🚀 GitHub Issue Tracker
## 📖 Overview
GitHub Issue Tracker is a modern web application that helps users track, manage, and organize GitHub issues in a clean and efficient way. It provides a user-friendly interface to view repository issues, filter them, and monitor progress without directly navigating GitHub.

This project is designed to improve productivity by simplifying issue tracking and providing a smooth user experience.

## ⚙️ Technologies Used

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* GitHub REST API
* Axios / Fetch API


## ✨ Features

* 🔍 Search GitHub repositories
* 📂 View all issues of a repository
* 🏷️ Filter issues by status (open/closed)
* ⚡ Fast and responsive UI
* 📱 Fully responsive design
* 🔄 Real-time data fetching from GitHub API

## 🛠️ How to Run Locally

Follow these steps to run the project on your local machine:

```bash
# 1. Clone the repository
git clone https://github.com/hydro1d/githubissuetrack.git

# 2. Navigate to the project folder
cd githubissuetrack

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Then open your browser and go to:
👉 [http://localhost:5173](http://localhost:5173)

Answer: map() transforms each element and returns a new array with the transformed values. filter() returns a new array containing only elements that satisfy a specified condition. forEach() iterates through elements for side effects without creating a new array.
